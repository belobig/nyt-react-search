# belobig's React NYT Scraper

* Search for articles
* Limit your search to 1, 5, or 10 articles
* Optionally limit your search by years
* Save articles

* This is a MERN app because it uses the following technologies, among others:
	- Mongoose and MongoDB
	- Express
	- React.js
	- Node.js
